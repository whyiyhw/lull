# Lull · 声音气象台

白噪音 + 生成式场景的**混音助眠/专注**网页应用。所有控制浮在一块随混音生长的画布上——你叠什么声音，画面就长出对应的气象（雨丝、林间浮尘、海浪、鸟鸣的金色火花、雷雨闪电、噪点颗粒…）。

**纯前端、零依赖、零构建**。双击 `index.html` 即可运行，无需联网、无需服务器。

---

## 运行

直接双击 `index.html`（或在浏览器里打开）。就这样。

## 功能

- **多声音叠加混音台**：任意叠加，每层独立音量 + 淡入淡出。
- **20 种声音 / 8 类**：噪音、雨、水、林、风、室内、暖、冥想。
- **混合音频引擎**：
  - 自然声（雨/瀑布/鸟鸣/森林/风/咖啡馆/火车…）用**真实录音**（`<audio>`）。
  - 纯噪音（白/粉/褐）、风扇/空调/低鸣、颂钵用**程序合成**（Web Audio，无限不循环、体积为零）。
  - 没有音频文件的声音会**自动回退到合成**，永远能用。
- **生成式场景**：整屏 canvas 随当前混音与音量实时作画；深/浅主题；尊重"减少动态"。
- **睡眠定时**：15/30/45/60/90 分，结束前 20 秒缓缓淡出（同时管真实与合成）。
- **床头时钟** + 当前混音诗行。
- 记住上次的音量、主题、每种声音的音量（localStorage）。

## 换 / 加音频（零改代码或一行）

1. 把音频文件（`.mp3`）放进 `audio/`，命名为对应 id：`lrain.mp3`、`storm.mp3`、`ocean.mp3`…
2. 在 `index.html` 的 `SOUNDS` 配置里给该声音加一个字段 `file:'xxx.mp3'` 即可（已有真实音的 8 个已加好）。
3. 没有 `file` 的声音自动用合成。

**推荐源**：[Pixabay](https://pixabay.com/sound-effects/)、[Freesound（CC0 过滤）](https://freesound.org/) —— 免版税、可商用、无需署名，覆盖全。

> ⚠️ **当前 `audio/` 里是探路用的占位素材**，部分是 CC BY-SA（要署名 + copyleft，**不适合商用**）。正式发布前请换成 CC0/可商用素材，详见 [AUDIO_CREDITS.md](AUDIO_CREDITS.md)。

## 技术

- 单文件 `index.html`：内联全部 CSS/JS，系统字体（`ui-serif` / `ui-monospace` / `system-ui`，零外链）。
- 合成：Web Audio（BufferSource 噪声 + BiquadFilter + LFO + ConvolverNode 混响；雷/鸟/篝火/颂钵用调度器与频率扫掠合成）。
- 真实音频：`<audio loop>` 元素，音量与主音量/定时统一管理。
- 场景：单个 `<canvas>`，按各声音的实时音量聚合出"气象强度"逐帧作画。
