# Lull · 声音气象台

白噪音 + 生成式场景的**混音助眠/专注**网页应用。所有控制浮在一块随混音生长的画布上——你叠什么声音，画面就长出对应的气象（雨丝、林间浮尘、海浪、鸟鸣的金色火花、雷雨闪电、噪点颗粒…）。

**本地优先、运行时零依赖、无账号、无埋点、无第三方请求**。可双击 `index.html` 运行，也可装到主屏离线使用。

---

## 运行

- **双击** `index.html` —— 直接用浏览器打开。合成声音全部可用；真实录音在 `file://` 下会因 `fetch` 受限**自动回退为合成音**（右下角有一次提示）。
- **本地服务器 / PWA**（推荐，能听真实录音、能离线、能锁屏播放）：
  ```bash
  pnpm install
  pnpm dev        # → http://localhost:5173
  ```
  或把整个目录部署到任意静态服务器 / GitHub Pages。浏览器地址栏会出现「安装」入口，装到主屏后可离线冷启动。

## 功能

- **多声音叠加混音台**：任意叠加，每层独立音量 + 淡入淡出。
- **26 种声音 / 8 类**：噪音、雨、水、林、风、室内、暖、冥想。
- **统一 Web Audio 引擎**（v0.2）：
  - 真实录音走 `fetch` + `decodeAudioData` 解码进 `AudioBufferSourceNode`（`loop=true`），与合成层**共用同一套 gain / 混响 / 主总线**——循环采样级无缝、iOS 音量推子生效、定时渐隐对所有层统一。
  - 纯噪音（白/粉/褐）、风扇/空调/低鸣、颂钵用**程序合成**（无限不循环、体积为零）。
  - 缺文件或 `file://` 下 fetch 失败的声音**自动回退合成**，永远能用。
- **调频拨盘 · 电台频道**（F-4）：复古 FM 拨盘（87.5–108 MHz），6 个内置频道固定刻度（88.1 雨夜 / 90.4 晨林 / 92.9 海边 / 95.6 炉边 / 98.3 车厢 / 100.9 深空），一键交叉淡化切换；拖动扫台有白噪嘶声 + 雪花反馈、松手吸附；可把当前混音「存为频道」（可拖动改频率、可删，上限 8）。`role=slider` 方向键可达。
- **锁屏存活**（F-2）：接入 Media Session（锁屏显示频道名与播放/暂停），静音保活元素防止后台挂起；**来电/闹钟等系统中断结束后自动重拉播放**（监听 `AudioContext.onstatechange` + 保活元素 `pause`，无需解锁亮屏，D7）。
- **会话恢复 / 分享**（F-5 / F-8）：记住上次混音，重开一键「继续上次」（尊重自动播放策略，不自动出声）；当前混音编码进 URL hash（`#lrain=60&fire=40`），点「Share」复制链接，打开即还原。
- **省电 / 停播熄火**（F-5）：页面切后台停画停钟、仅保留音频调度；无高频动效时画面自适应降至 ~30fps；**暂停/定时结束后挂起 AudioContext，引擎不空转**（D8）。
- **沉浸挂机模式**（F-9）：播放中 8 秒无操作淡出控制台与光标、放大时钟；`F` 全屏；Screen Wake Lock 播放时保持亮屏；定时启动后画面渐暗至约 30%（入睡调光）；时钟每分钟微漂移防烧屏；粒子密度按屏幕面积缩放；快捷键 `Space` 播放暂停 / `F` 全屏 / `Esc` 退出沉浸 / `1–6` 切内置频道；点时钟切换秒针。
- **细分子选择器**：把一个声音再拆成可勾选的成员——鸟鸣挑鸟种（麻雀/画眉/斑鸠/杜鹃/夜莺）、虫鸣挑虫种（蟋蟀/蝉/蛙/纺织娘）、篝火挑爆裂性格（噼啪/爆裂/塌陷/余烬）、咖啡馆挑环境细节（碰杯/磨豆/咖啡机/汤匙）——以上多选合唱；雨可选「打在什么上」（屋檐/窗玻璃/树叶/水面，单选实时换质地，对小雨/大雨/雷雨通用）。选中即开对应声音、随其推子发声、本地持久化。
- **生成式场景**：整屏 canvas 随当前混音与音量实时作画；深/浅主题；尊重「减少动态」。
- **睡眠定时**：15/30/45/60/90 分，结束前 20 秒缓缓淡出（统一走主总线）。
- **床头时钟** + 当前混音诗行。
- **PWA**：manifest + service worker，可安装、可离线（含音频素材缓存）。
- 记住上次的音量、主题、每种声音的音量（localStorage）。

## 换 / 加音频（零改代码或一行）

1. 把音频文件（`.mp3`）放进 `audio/`，命名为对应 id：`lrain.mp3`、`storm.mp3`、`ocean.mp3`…
2. 在 `index.html` 的 `SOUNDS` 配置里给该声音加一个字段 `file:'xxx.mp3'` 即可（当前 `forest`/`train`/`cafe` 三个已挂真实录音）。
3. 没有 `file` 的声音自动用合成。
4. 若新增了文件，记得把文件名同步进 `sw.js` 的 `AUDIO` 列表，离线才会缓存到。

> ⚠️ 引擎用 `AudioBufferSourceNode` 采样级循环，**素材必须首尾可循环**（首尾能量/相位接近），否则循环点会跳变。
>
> **一步到位的预处理**：把原始录音丢进 `audio/raw/`，运行 `pnpm prep-audio`（依赖系统 `ffmpeg`）——脚本自动**转单声道 → 首尾交叉淡化消接缝 → loudnorm 响度归一 → 转码 mp3**，产出直接落到 `audio/`。单文件也可：`node scripts/prep-audio.mjs 下载.wav lrain`。
>
> **推荐源**：[Pixabay](https://pixabay.com/sound-effects/)、[Freesound（CC0 过滤）](https://freesound.org/) —— 免版税、可商用、无需署名。
>
> ✅ **当前发布集全部可商用**：仅 `forest`/`train`/`cafe` 三个 CC0/公有领域录音 + 其余 CC0 程序合成；原 5 个 CC BY-SA 素材已移除改用合成（F-3 合规）。详见 [AUDIO_CREDITS.md](AUDIO_CREDITS.md)。

## 技术

- 单文件 `index.html`：内联全部 CSS/JS，系统字体（`ui-serif` / `ui-monospace` / `system-ui`，零外链）。
- 音频：Web Audio 统一图 —— 合成层（BufferSource 噪声 + BiquadFilter + LFO + ConvolverNode 混响；雷/鸟/篝火/颂钵用调度器合成）与真实录音层（`decodeAudioData` → `AudioBufferSourceNode`）共用 gain / 混响 / 主总线。
- 锁屏：Media Session API + 静音 WAV 保活元素（与 `mediaSession` 特性检测解耦）+ `AudioContext.onstatechange`/保活元素 `pause` 自愈中断（D7）+ `visibilitychange`/`pageshow` 兜底。停播 0.9s 后 `ctx.suspend()` 熄火（D8）。
- 场景：单 `<canvas>`，按各声音实时音量聚合「气象强度」逐帧作画；后台停画、无高频动效降帧；粒子数按屏面积缩放（设上限）。
- 挂机：Screen Wake Lock（仅「播放且可见」持有）+ 8s 空闲隐藏 UI + 全屏 API + 入睡调光遮罩 + 时钟防烧屏漂移。
- 离线：`sw.js`（壳 network-first、音频/图标 cache-first）+ `manifest.webmanifest`。
- 图标：`scripts/gen-icons.mjs` 纯 Node 生成（无第三方依赖），源图见 `icons/icon.svg`。
